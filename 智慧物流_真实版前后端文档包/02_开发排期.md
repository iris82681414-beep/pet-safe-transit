### 后端共同任务

- Spring Boot 主项目创建。

- 建立统一包结构：`auth / vehicle / cargo / tracking / alert / command / device / assistant / common`。
- 配置统一响应、统一异常、参数校验。
- 配置 Swagger/Knife4j，调试入口为 `http://localhost:8080/doc.html`。

### 后端 1

- PostgreSQL 基础库连接。
- users / vehicles / cargo / cargo_vehicle_binding 建表。
- seed 演示账号和车辆货物。
- 完成 `/auth/login`、`/users/me`。

### 后端 2

- EMQX 启动与连接测试。
- Kafka 启动，创建 `gps-points`、`vehicle-heartbeats`。
- TimescaleDB 启动，创建 `gps_points`。
- Redis 连接测试。
- 写 MQTT 订阅空壳。

### 后端 3

- MinIO 启动。
- pgvector 扩展确认。
- alerts / commands / devices / knowledge_documents 建表。
- docker-compose 初版。

### 前端 1

- Vue 3 + Vite + TS 项目初始化。
- 接入 Leaflet / Axios / Pinia / Naive UI。
- 迁移货物追踪页、车辆总览页结构。

### 前端 2

- 登录页、告警中心、车辆管理页基础路由。
- Axios 请求封装、Token 保存、路由守卫。

### 当天验收

```txt
1. docker-compose 能启动 PostgreSQL/TimescaleDB/Redis/EMQX/Kafka/MinIO。
```
