## 10. 遇到的问题和解决方式

### 问题 1：PostgreSQL 驱动找不到

报错：

```txt
ClassNotFoundException: org.postgresql.Driver
```

解决：

- 在 `pom.xml` 中加入 PostgreSQL JDBC 驱动。
- 检查 `application-dev.yml` 中的数据库连接配置。

### 问题 2：Mapper Bean 冲突

报错：

```txt
ConflictingBeanDefinitionException: logisticsVehicleMapper
```

原因：

- 旧包和新包里同时存在同名 `LogisticsVehicleMapper`。

解决：

- 清理旧的苍穹外卖相关 Mapper。
- 保留 `com.sky.logistics.mapper.LogisticsVehicleMapper`。

### 问题 3：端口被占用

报错：

```txt
Port 8080 was already in use
```

解决：

- 停止旧的 Spring Boot 进程。
- 再重新启动当前项目。

### 问题 4：车辆表设计不正式

问题：

- 旧版本车辆表没有独立 `id` 主键。
- 绑定关系直接使用车牌，不适合作为正式后端设计。

解决：

- 给 `vehicles` 增加 `id BIGSERIAL PRIMARY KEY`。
- 绑定表改成 `vehicle_id BIGINT` 外键。
- 保留 `plate` 作为展示字段，避免业务和数据库主键混用。
