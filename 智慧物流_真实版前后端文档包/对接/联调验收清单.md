# 联调验收清单

每天晚上跑一遍。只要主链路不通，第二天先修主链路。

## Day 1 验收

- [ ] 前端能启动。
- [ ] 后端能启动。
- [ ] Swagger/Knife4j 能打开：`http://localhost:8080/doc.html`，能看到 `智慧物流接口` 分组。
- [ ] PostgreSQL/TimescaleDB 能连。
- [ ] Redis 能连。
- [ ] EMQX Dashboard 能打开。
- [ ] Kafka topic 能创建。
- [ ] MinIO Console 能打开。

## Day 2 验收

- [ ] 在 Swagger/Knife4j 中调用 `POST /api/v1/auth/login` 成功，并拿到 `data.accessToken`。
- [ ] `GET /vehicles` 有数据。
- [ ] `POST /cargo/bind` 能绑定。
- [ ] MQTTX publish `vehicle/沪A-C0291/gps`。
- [ ] 后端日志显示收到 MQTT。
- [ ] Kafka `gps-points` 有消息。
- [ ] TimescaleDB `gps_points` 有数据。
- [ ] Redis 有车辆最新位置。
- [ ] `/cargo/{cargoId}/position` 返回 MQTT 上报坐标。

## Day 3 验收

- [ ] 前端地图展示真实位置。
- [ ] WebSocket 能收到 `vehicle.position`。
- [ ] `/trajectory` 返回多个点。
- [ ] `/eta` 返回 ETA 和进度。
- [ ] 车辆管理能新增车辆。
- [ ] 告警列表能显示。

## Day 4 验收

- [ ] MQTT heartbeat 上报后设备在线。
- [ ] 超过 90 秒未上报能离线。
- [ ] 偏航能触发告警。
- [ ] 告警能确认。
- [ ] 告警能关闭。
- [ ] 调度指令能下发到 MQTT。
- [ ] command/ack 回来后状态变成 EXECUTED。

## Day 5 验收

- [ ] MinIO 能上传知识库文档。
- [ ] pgvector 有 chunk embedding。
- [ ] `/assistant/chat` 返回 answer 和 sources。
- [ ] 5 个角色页面都能打开。
- [ ] 主流程能完整跑通一次。

## Day 6 验收

- [ ] docker-compose 一键启动。
- [ ] README 可执行。
- [ ] 演示账号可用。
- [ ] MQTTX 样例消息可用。
